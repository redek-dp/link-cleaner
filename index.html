Este atalho adiciona o Link Cleaner ao menu de compartilhamento do seu dispositivo quando você compartilha um aplicativo. Deste jeito, você pode enviar links primeiro para o Link Cleaner, depois usar o botão Compartilhar do Link Cleaner para enviar o novo link para o destino pretendido.

Você pode ver um aviso sobre atalhos não confiáveis. Para permitir atalhos não confiáveis, abra o Configurações, toque no aplicativo Atalhos e ative "Permitir atalhos não confiáveis".
